Colocar documentario pra executar local

# Como rodar front-react:

Você irá precisar do <a href="https://nodejs.org/en/">Nodejs e do Npm</a>

### Acesse a pasta

```
cd <profise-front>
```

### Instale as dependencias

```
npm i
```

### Start na aplicacao

```
npm run dev
```

### Port:

```
http://localhost:5173/
```
